    InputOptions

    Contains rules for validating and/or filtering an input.
    Properties:
    Name Type Default Description
    name String '' Input's name.
    required Boolean false Whether input is required or not.
    filters Array [ Any filter functions to apply to value.
    validators Array [ Any validators to validate against for given value (to validator).
    breakOnFailure Boolean false Whether or not to 'break' on a validation failure result or not.
    valueObscured Boolean false Whether to obscure the value being tested against (to the assigned places) or not).
    valueObscurator function ((x) => x) - Obscurator used for obscuring a value given to validation.
