    Properties:
    Name Type Default Description
    name String `Input` this result was generated with.
    result Boolean Result of validation.
    messages Array Any error messages if `result` is `false`.
    value * null Value tested against (if `filters` exist on given `#Input` object the `value` is what is returned from the results of running filters on value).
    rawValue * null Raw value tested against.
    obscuredValue * null Obscured value if validation `result` is `true`.
    filteredValue * null Filtered result if validation `result` is `true`.
